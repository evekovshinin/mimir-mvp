🏗️ ARCHITECTURE
================================================================================
3-Layer Architecture:

    CLI Layer (Typer)
    ├─ Commands: create-project, projects, create-task, commit, branch, switch, history, show, status
    ├─ Rich formatting for beautiful output
    └─ Error handling with user-friendly messages

    Service Layer
    ├─ ProjectService: project management (hierarchical)
    ├─ TaskService: task management
    ├─ CommitService: commit operations, history, merge
    └─ BranchService: branch management

    ORM Layer (SQLAlchemy)
    ├─ Project model (hierarchical)
    ├─ Task model (belongs to project)
    ├─ ContextCommit model (immutable)
    ├─ CommitParent model (DAG edges)
    └─ Branch model (pointers)

    Database (PostgreSQL)
    ├─ tasks table
    ├─ context_commits table
    ├─ commit_parents table (supports merge)
    └─ branches table

🗄️ DATABASE SCHEMA
================================================================================
Tables:
- projects (id, name, parent_id, created_at) — hierarchical structure
- tasks (id, project_id, name, external_id, created_at)
- context_commits (id, task_id, message, full_context, author, cognitive_load, uncertainty, created_at)
- commit_parents (child_id, parent_id) — supports merge
- branches (id, task_id, name, head_commit_id, created_at)

Constraints:
- UNIQUE(projects.name)
- UNIQUE(projects.parent_id, projects.name) — nested unique
- UNIQUE(tasks.project_id, tasks.name)
- UNIQUE(tasks.external_id) — if not null
- UNIQUE(branches.task_id, branches.name)
- UNIQUE(commit_parents.child_id, commit_parents.parent_id)
- Foreign key relationships with cascade deletes
- Performance indexes on frequently queried columns

Features:
- DAG structure with CommitParent
- Immutable ContextCommit
- Branch as pointer to commit
- Support for merge (multiple parents)
- Recursive CTE for efficient history traversal

🎮 CLI COMMANDS
================================================================================
mimir create-task <name>                    Create new task with main branch
mimir commit [options]                      Create commit with context
  --task, --branch, --message, --context-file
  --cognitive-load, --uncertainty, --author

mimir branch [list|create|delete] [options] Manage branches
  create <name> --task --from
  delete <name> --task
  list --task

mimir switch --task --branch               Switch current task/branch

mimir history --task --branch [--limit]     Show commit history (recursive CTE)

mimir show <commit-id>                      Show full context of commit

mimir status                                Show current task and branch

mimir init                                  Initialize database
